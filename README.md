# Evaluador de Notas en JavaScript

Programa en JavaScript que evalúa la calificación de un estudiante y muestra un mensaje según el rango de la nota.

## Rangos de Evaluación
- 90 a 100 → Excelente  
- 75 a 89 → Bien  
- 60 a 74 → Suficiente  
- Menor a 60 → No aprobado
